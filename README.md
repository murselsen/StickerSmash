# StickerSmash

Photo Editing Application with React Native & Expo

A cross-platform mobile application that allows you to select photos from your device gallery, decorate them with emojis (stickers), and save your edited images to your device.

## Features

- 🖼️ **Gallery Access**: Select images from device library
- 🎭 **Emoji/Sticker Support**: Drag-and-drop stickers onto photos
- 💾 **Save**: Export edited photos to local gallery
- 📱 **Cross-Platform**: Full compatibility on Android, iOS, and Web
- 🎨 **Modern UI**: Dark mode support, clean user interface

## Technologies

- React Native & Expo
- Expo Image Picker
- Expo Media Library
- React Native Gesture Handler (For sticker interactions)
